Ubuntu CUDA Install (Ubuntu 18.04)

前言

本篇將介紹Ubuntu 18.04安裝CUDA

雖然NVIDIA CUDA Toolkit裡面有包含GPU的驅動程式，但是還是建議手動另外安裝驅動程式。
以下範例以下列環境作為範例：

OS：Uubuntu 18.04 x64
GPU：RTX2080Ti
CUDA：NVIDIA CUDA 10.2 Tookit

開始安裝

前置作業

Disable Nouveau Nvidia driver on Ubuntu

Confirm the content of the new modprobe config file

安裝編譯器

NVIDIA驅動下載安裝

NVIDIA驅動下載

查了一下CDUA Toolkit Release Note，NVIDIA CUDA 10.2 支援的NVIDIA Driver版本為440.3
我們只要找大於440.3版本的驅動就可以了

再到NVIDIA 驅動程式下載，找到相對應的驅動程式下載就好

NVIDIA驅動安裝

抓下來會是一個.run檔，切換到下載的路徑，在執行這個檔案跟著指示安裝就完成了

測試NVIDIA驅動

輸入以下指令，如果有跑出下列畫面表示驅動程式有安裝完成

NVIDIA CUDA下載安裝

NVIDIA Driver裝好後接著就可以安裝CUDA

NVIDIA CUDA下載

到NVIDIA CUDA下載頁面下載CDUA

依照NVIDIA給的指令，下載CUDA

NVIDIA CUDA安裝

開始安裝

這邊要注意的是，因為我們已經有手動安裝NVIDIA Driver所以當出現以下提示時選”n”，其跟著指示安裝就完成了
